ArchVision AI Home Builder is a powerful, full-stack application I developed to demonstrate the transformative potential of multimodal Generative AI in the architectural and real estate industries. 

It functions as an instant, intelligent design engine, taking complex user specifications (style, size, features) and instantly rendering two essential outputs:Photorealistic Exterior Renders (for visualization and marketing)Technical Blueprints/Floor Plans (for planning and feasibility)This project showcases expertise in integrating advanced AI models (like Gemini) with a modern frontend architecture to create a delightful, feature-rich user experience.Key Technical Highlights & Skills DemonstratedCategorySkills & Tools UsedValue PropositionGenerative AI & LLMsGoogle Gemini API (@google/genai), Function Calling, Structured Data SynthesisAbility to translate complex, natural language requirements into structured, executable prompts and multimodal outputs (images + analysis).
